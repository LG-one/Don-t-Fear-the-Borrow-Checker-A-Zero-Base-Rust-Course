# 🦀 Rust 基础课程 v3.0

> 从 "Hello World" 到写一个完整的命令行记事本，15 课搞定 Rust。

## 这是什么？

一套给**完全没碰过 Rust**的人准备的入门课。

不是那种"我给你列个知识点你自己看"的课——每节课都有一个**真正能跑的项目**：温度转换器、猜数字游戏、扑克牌发牌器、学生成绩管理、并发文件处理器……学到最后你会自己写一个带 JSON 持久化的命令行记事本。

每节课末尾有 3 道练习题，还有 #[test] 测试可以玩。写完代码跑一下测试，绿了就是会了，红了就再看看。

## 怎么跑？

```bash
# 装 Rust（没装的话）
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# 跑任意一课
cargo run --example lesson00    # Hello World
cargo run --example lesson01    # 变量和类型
cargo run --example lesson14    # 最终实战：命令行记事本

# 跑测试（先取消练习题区域的注释）
cargo test --example lesson04

# 异步编程课外资料（需要 tokio）
cargo run --example lesson_async --features async_support
```

## 几个小建议

1. **别跳课** — 后面的课会用到前面的知识，跳了会懵
2. **别光看** — 代码要自己敲，运行，改，再运行
3. **别怕报错** — Rust 编译器是你见过最友好的编译器，它会告诉你哪里错了、为什么错、怎么改
4. **别急** — 09、10 课内容多，分两天学完全没问题
5. **做练习** — 练习题不是摆设，是真的帮你巩固的

## 需要什么？
